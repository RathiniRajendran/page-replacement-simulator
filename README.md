# Page Replacement Simulator

## Project Overview
This project is a web-based simulator that demonstrates how different **Page Replacement Algorithms** work in an Operating System.

The simulator takes a **reference string** and **number of frames** as input, runs multiple algorithms, and compares their performance based on **page faults** and **page hits**.

This project was developed for the **Operating Systems (EC 6110) Assignment**.

## Implemented Algorithms

The simulator implements the following algorithms:

- FIFO (First In First Out)
- LRU (Least Recently Used)
- LFU (Least Frequently Used)
- MFU (Most Frequently Used)

Each algorithm processes the same reference string and frame capacity.


## Features

- User input for number of frames
- User input for reference string
- Simulation of multiple page replacement algorithms
- Automatic calculation of page faults and hits
- Comparison of algorithm performance
- Identification of the best algorithm
- Step-by-step visualization of memory frames
